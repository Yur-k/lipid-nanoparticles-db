Nanoparticle Protein Corona JSON Schema Documentation

This JSON schema defines the structure for a nanoparticle protein corona dataset.

Properties

  • type (string, required): The chemical composition of the nanoparticle.
  • mean_hydrodynamic_diameter (number or null, optional): The mean hydrodynamic diameter of the nanoparticle.
  • zeta_potential (number or null, optional): The zeta potential of the nanoparticle.
  • pdi (number or null, optional): The polydispersity index of the nanoparticle.
  • article (object, required): The article information associated with the nanoparticle dataset.
    • article_doi (string, required): The DOI of the article.
    • title (string, required): The title of the article.
    • protein_identification_method (string, required): The protein identification method used.
  • coronas (array, required): The protein corona information for the nanoparticle.
    • mean_hydrodynamic_diameter (number or null, optional): The mean hydrodynamic diameter of the nanoparticle.
    • zeta_potential (number or null, optional): The zeta potential of the nanoparticle.
    • pdi (number or null, optional): The polydispersity index of the nanoparticle.
    • exposure_time (number or null, optional): The exposure time of the experiment in minutes.
    • protein_source (string or null, optional): Source of protein.
    • experiment_type (string, required): The type of experiment conducted: in vivo/in vitro.
    • proteins (array, required): The list of proteins found in the protein corona.
      • name (string, required): Protein or part of protein.
      • rpa (number or null, optional): The relative protein abundance of the protein in the corona.

JSON Example


{
    "mean_hydrodynamic_diameter": 103.2,
    "pdi": null,
    "type": "DOTAP:DC-Chol:DOPC:DOPE:PEG5000",
    "zeta_potential": 11.3
    "protein_identification_method": "LC-MS-MS",
    "article": {
        "article_doi": "10.1039/c3nr05559k",
        "title": "Example of title"
    },
    "coronas": [
        {
        "experiment_type": "in vitro",
        "exposure_time": 60,
        "mean_hydrodynamic_diameter": 111,
        "protein_source": "Mouse Plasma",
        "zeta_potential": -7
        "pdi": null,
        "proteins": [
            {
            "name": "Ig kappa chain C region",
            "rpa": 7.868
            },
            {
            "name": "Complement C3",
            "rpa": 4.639
            }
        ],
        }
    ],
}
